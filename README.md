This is a Hardware Project.
The purpose of this project is to convert Text input given from keyboard to Braille signals . 
coin motors form a grid of 3X1.
the grid is operated from the method created in the code. the dictionary in the code is where all letters are mapped to specific vibration and unique grid motor pattern.
the uniqueness of this project lies in shrinking down the general braille of 3X2 grid to 3X1.
